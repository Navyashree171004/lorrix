# Lorrix – Logistics & LR Management System

A modern product website for Lorrix by ZYP Logistech.

Built with Vite + React + TypeScript + Tailwind CSS + shadcn/ui.

## Getting Started

```sh
npm install
npm run dev
```

## Build for production

```sh
npm run build
```

## Tech Stack

- Vite
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
